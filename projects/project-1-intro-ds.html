Year 1 Block A

📊 Introduction to Data Science

My first exploration into the world of AI and Data Science, building foundational skills in data visualization, Python programming, and mathematical concepts that form the backbone of machine learning.

🐍 Python 📊 Power BI 📐 Mathematics 🔢 NumPy

Project Overview

This introductory block marked my entry into the Data Science & AI program at Breda University of Applied Sciences. The project focused on understanding the fundamentals of data science workflows, from data collection to visualization and insight generation.

Working with real-world datasets, I learned to transform raw data into meaningful visualizations that tell a story and support data-driven decision making.

Key Components

  • Introduction to AI & Data Science: Understanding the landscape of artificial intelligence, machine learning paradigms, and the role of data scientists in modern organizations.
  • Data Visualization with Power BI: Creating interactive dashboards and reports to communicate insights effectively to stakeholders.
  • Python Programming Fundamentals: Learning the syntax, data structures, and programming paradigms essential for data manipulation and analysis.
  • Mathematical Foundations: Building intuition for statistics, probability, and linear algebra concepts used throughout the program.

Skills Developed

This project established my foundation in the CRISP-DM methodology—the systematic approach to data science project management that I continue to apply in all subsequent work.

  • Data cleaning and preprocessing techniques
  • Exploratory data analysis (EDA)
  • Creating effective data visualizations
  • Basic statistical analysis and interpretation
  • Version control with Git

Learning Outcomes

  • Analyze and describe data science problems within their business context
  • Apply appropriate visualization techniques to communicate insights
  • Write clean, readable Python code following best practices
  • Apply mathematical concepts to solve data-related problems
