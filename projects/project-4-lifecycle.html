Year 1 Block D

🔄 Data Science Lifecycle Management

The capstone of Year 1—a team-based multidisciplinary project bringing together all learned skills. Applying agile methodologies, time series analysis, and data engineering practices while collaborating with stakeholders.

📋 Agile/Scrum 📈 Time Series ⚙️ Data Engineering 👥 Stakeholder Mgmt

Project Overview

This final block of Year 1 synthesized everything learned throughout the first year into a comprehensive team project. Working with fellow students, we tackled a real-world data science challenge from problem definition through deployment.

The focus shifted from individual technical skills to collaborative development, stakeholder communication, and managing the complete data science lifecycle.

Key Components

  • Agile Project Management: Running sprints, daily standups, sprint reviews, and retrospectives. Using tools like Jira and Trello for task management.
  • Time Series Analysis: Working with temporal data, understanding seasonality, trends, and building forecasting models.
  • Data Engineering: Building robust data pipelines, ETL processes, and ensuring data quality throughout the workflow.
  • Stakeholder Management: Regular presentations, gathering requirements, and translating business needs into technical specifications.

Team Collaboration

This project emphasized the collaborative nature of real-world data science:

  • Worked in a cross-functional team of 4-5 students
  • Practiced code reviews and collaborative Git workflows
  • Held weekly stakeholder presentations and feedback sessions
  • Documented decisions and maintained project wikis
  • Managed scope and priorities through sprint planning

Technical Implementation

The project required integrating multiple technical components:

  • Built automated data ingestion pipelines
  • Implemented time series forecasting (ARIMA, Prophet)
  • Created interactive dashboards for stakeholder reporting
  • Developed documentation and handover materials
  • Applied version control best practices for team development

Learning Outcomes

  • Apply agile methodologies to manage data science projects
  • Analyze and forecast time series data using appropriate techniques
  • Build maintainable data engineering pipelines
  • Communicate effectively with technical and non-technical stakeholders
  • Collaborate effectively in a team environment
