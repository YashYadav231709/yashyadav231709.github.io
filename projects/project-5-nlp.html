Year 2 Block A

💬 Natural Language Processing

Diving into the world of text and speech processing—building speech-to-text systems, text classification models, and exploring the transformer architecture that powers modern language models.

🤗 Transformers 📝 NLP 🎙️ Speech-to-Text ✨ Generative AI

Project Overview

Year 2 began with an intensive exploration of Natural Language Processing. This project covered the full spectrum of NLP tasks, from classical text processing to modern transformer-based approaches and generative AI applications.

The skills developed here directly connect to my current internship work, where I build transcription and NLP analysis pipelines for video content.

Key Components

  • Speech-to-Text: Building automatic speech recognition (ASR) systems, understanding acoustic models, and working with audio data preprocessing.
  • Translation: Exploring sequence-to-sequence models and the challenges of machine translation.
  • Text Classification: Sentiment analysis, topic classification, and document categorization using various approaches.
  • Transformer Models: Deep dive into attention mechanisms, BERT, GPT architectures, and fine-tuning pre-trained models.
  • Generative AI: Introduction to language generation, prompt engineering, and responsible use of generative models.

Technical Implementation

Throughout this project, I worked with cutting-edge NLP tools and frameworks:

  • HuggingFace Transformers library for model fine-tuning
  • spaCy for text preprocessing and entity recognition
  • Whisper and other ASR models for speech transcription
  • NLTK for classical NLP tasks
  • Building custom tokenizers and text processing pipelines

Projects Completed

Key deliverables from this block included:

  • A sentiment analysis system for customer reviews
  • Speech transcription pipeline with speaker diarization
  • Text summarization using transformer models
  • Named entity recognition for information extraction
  • Exploration of prompt engineering techniques

Learning Outcomes

  • Build and deploy speech-to-text transcription systems
  • Apply transformer models to various NLP tasks
  • Understand the architecture and training of language models
  • Implement text classification and information extraction pipelines
  • Work responsibly with generative AI technologies
