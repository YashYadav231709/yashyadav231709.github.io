Year 1 Block B

🤖 Machine Learning

Deep dive into classical machine learning algorithms, understanding the mathematical foundations behind supervised and unsupervised learning while building practical skills in model development and evaluation.

🔬 Scikit-learn 🗄️ SQL 📐 Linear Algebra 📈 Calculus

Project Overview

This block introduced me to the core algorithms that power machine learning systems. From linear regression to decision trees, I learned not just how to implement these algorithms but why they work—grounded in calculus and linear algebra.

The project involved building predictive models on structured datasets, with emphasis on proper model evaluation, cross-validation, and avoiding common pitfalls like overfitting.

Key Components

  • Data Analysis with Python: Advanced Pandas operations, feature engineering, and data preprocessing pipelines for ML-ready datasets.
  • Machine Learning Algorithms: Implementation of regression, classification, and clustering algorithms using Scikit-learn.
  • Calculus & Linear Algebra: Understanding gradient descent, matrix operations, and the mathematics behind optimization.
  • SQL Database Integration: Querying and managing data in relational databases, connecting ML pipelines to persistent storage.

Algorithms Explored

During this project, I implemented and compared various ML algorithms:

  • Linear and Logistic Regression
  • Decision Trees and Random Forests
  • K-Nearest Neighbors (KNN)
  • Support Vector Machines (SVM)
  • K-Means Clustering
  • Principal Component Analysis (PCA)

Technical Highlights

A significant portion of this project focused on understanding the bias-variance tradeoff and developing intuition for hyperparameter tuning. I learned to use cross-validation effectively and interpret metrics like precision, recall, and F1-score in context.

  • Built end-to-end ML pipelines with preprocessing steps
  • Implemented custom evaluation metrics
  • Created SQL queries for feature extraction
  • Applied regularization techniques (L1, L2)

Learning Outcomes

  • Select and apply appropriate ML algorithms for different problem types
  • Evaluate model performance using proper validation techniques
  • Understand the mathematical foundations of optimization
  • Design and query relational databases for ML workflows
