Year 2 Block B

👁️ Computer Vision & Robotics

From pixels to perception—building object detection and image segmentation systems while exploring the fundamentals of robotics, control theory, and reinforcement learning.

📷 OpenCV 🎯 YOLO 🤖 Robotics 🎮 Reinforcement Learning

Project Overview

This block expanded on deep learning foundations to tackle computer vision challenges. I learned both traditional image processing techniques and modern deep learning approaches for visual understanding.

The robotics component introduced control systems and reinforcement learning, showing how AI systems can interact with and learn from physical environments.

Key Components

  • Traditional Computer Vision: Image filtering, edge detection, feature extraction (SIFT, ORB), and classical computer vision algorithms.
  • Object Detection: Training and deploying detection models (YOLO, Faster R-CNN) for real-time object recognition.
  • Image Segmentation: Semantic and instance segmentation using U-Net, Mask R-CNN, and other architectures.
  • Control Theory Fundamentals: Understanding PID controllers, state-space models, and feedback systems.
  • Reinforcement Learning: Q-learning, policy gradients, and training agents in simulated environments.

Technical Implementation

This project involved extensive hands-on work with visual data:

  • OpenCV for image preprocessing and augmentation
  • Custom dataset annotation using LabelImg and CVAT
  • Training YOLO models on domain-specific datasets
  • Implementing segmentation models with PyTorch
  • Working with video streams and real-time inference

Robotics & RL Exploration

The robotics component provided insight into embodied AI:

  • Simulated robot control using Gym environments
  • Implemented basic RL algorithms (DQN, A2C)
  • Explored sim-to-real transfer challenges
  • Understanding sensor fusion and perception systems
  • Built simple autonomous navigation behaviors

Learning Outcomes

  • Apply both classical and deep learning approaches to visual tasks
  • Train and deploy object detection models for real-world applications
  • Implement image segmentation for scene understanding
  • Understand fundamentals of control systems and robotics
  • Apply reinforcement learning to sequential decision problems
